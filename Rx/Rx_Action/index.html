A unit of work to be executed in a `scheduler`. An action is typically created from within a @link SchedulerLike and an RxJS user does not need to concern themselves about creating and manipulating an Action.

Schedules this action on its parent @link SchedulerLike for execution. May be passed some context object, `state`. May happen at some point in the future, according to the `delay` parameter, if specified.

parameter state

Some contextual data that the `work` function uses when called by the Scheduler.

parameter delay

Time to wait before executing the work, where the time unit is implicit and defined by the Scheduler.
